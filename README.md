# 🔍 MISFIT — Daily Logic Challenge

**MISFIT** is a minimalist web-based puzzle game designed for professionals. Every day, users are challenged to identify the "intruder" in 5 different corporate-themed sets.

[▶️ Play MISFIT now](https://NasrCH.github.io/misfit-game)

---

## 🚀 Features
- **Daily Ritual:** 5 new hand-picked challenges every 24 hours.
- **Speed-Based Scoring:** The faster you find the Misfit, the higher your score.
- **LinkedIn Ready:** One-click share feature with emoji-based results.
- **Minimalist UI:** Designed with LinkedIn's professional aesthetic in mind.

## 🛠️ Tech Stack
- **Language:** Vanilla JavaScript (ES6+)
- **Styling:** CSS3 Custom Properties (LinkedIn Color Palette)
- **Hosting:** GitHub Pages

## 📖 How to Play
1. Read the **Topic** for the current round.
2. Identify the one term that **doesn't belong** in the set.
3. Tap it before the timer runs out!
